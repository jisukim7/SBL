# SBL

SBL is a programming language implemented with Python using the SLY library.

**Installation Instructions for SLY 0.4:**

Using PIP:

       $ pip3 install sly
or

       $ python3 -m pip install sly


**SBL Datatypes:**

  - Numbers (Positive/negative integer or real)
  - Booleans
  - Strings
  - List
  - Tuple
